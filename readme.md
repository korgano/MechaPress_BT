# Mecha-Press BattleTech Content

All credit belongs to the original authors and artists.

This repo serves as an online resource for accessing the BattleTech related Technical Readouts, game rule proposals, fanfic, and articles from the 1990s Mecha-Press magazines.

These are not only interesting historical artifacts, but also useful inspiration for anyone thinking up custom BattleTech content.
